# 安装与环境部署

## 安装 Vivado

Vivado Design Suite 是 Xilinx 为 HDL 设计的综合和分析而生产的软件套件，我们会利用 Vivado 编写设计我们的 MIPS 指令集等项目。本实验由于开发板限制，建议使用 2017.2 版本。

Vivado 受到出口管制的限制，需要首先在官网注册账户，才能下载使用。注册账户之后，我们进入 [Xilinx | Vivado archive](https://www.xilinx.com/support/download/index.html/content/xilinx/en/downloadNav/vivado-design-tools/archive.html)，点击 2017.2，选择 Vivado HLx 2017.2: WebPACK and Editions - Windows Self Extracting Web Installer 下载。
